# 《茶经》抓取工具

这是一个用于抓取古诗文网上《茶经》文本的Python脚本。《茶经》是唐代陆羽所著，被誉为世界上第一部茶叶百科全书。

## 内容简介

《茶经》是中国乃至世界现存最早、最完整、最全面介绍茶的第一部专著，被誉为茶叶百科全书。此书是关于茶叶生产的历史、源流、现状、生产技术以及饮茶技艺、茶道原理的综合性论著，是划时代的茶学专著，精辟的农学著作，阐述茶文化的书。将普通茶事升格为一种美妙的文化艺能，推动了汉族茶文化的发展。

## 使用方法

1. 安装依赖:
   ```
   pip install -r requirements.txt
   ```

2. 运行脚本:
   ```
   python scrape_chajing.py
   ```

3. 抓取的内容将保存在 `chajing` 目录下，每个章节一个文件，并创建一个完整合并版本

## 内容结构

《茶经》共分十章:
- 一之源：阐述茶的起源、生长环境等
- 二之具：茶的采摘工具和方法
- 三之造：制茶工艺
- 四之器：茶具
- 五之煮：煮茶方法
- 六之饮：品茶方法
- 七之事：历史上关于茶的记载
- 八之出：茶的产地
- 九之略：简略部分，讲述饮茶的礼节
- 十之图：茶具图示
